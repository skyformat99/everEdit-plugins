# EverEdit-plugins

*  这里发布一些个人插件,按以下格式发布
*  除了说明文件以外,其他的代码或图片文件都放在对应的文件夹中.
需要安装某一扩展的时候,只需要把对应的文件夹下的文件或文件夹复制粘贴到 EverEdit 根目录,选择合并文件夹,重启 EverEdit 或 刷新扩展菜单,就可以使用了
