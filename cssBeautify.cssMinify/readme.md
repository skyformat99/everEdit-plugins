## cssBeautify.ejs

实现格式化 CSS 功能 
虽然CSS Mode 已经默认包含了这个功能  但是这里的实现方式更简单,效率更高

同样是从站长之家 http://tool.chinaz.com/Tools/CssFormat.aspx 页面扒下来的代码

可配置是否横向排列 horizontal 默认为 false

     当 horizontal = true:
	    body {padding-top:20px;padding-bottom:20px}
    .navbar {margin-bottom:20px}
	
     当 horizontal = false:
	    body {
			padding-top:20px;
	    	padding-bottom:20px
		}
		.navbar {
			margin-bottom:20px
		}
